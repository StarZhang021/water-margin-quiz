# 水浒人物性格测评

基于《水浒传》一百单八将性格特质的趣味测评系统。

## 功能特色

- **双版本选择**：24题精简版（约3分钟）/ 96题完整版（约10分钟）
- **八维度测评**：侠义心肠、沉稳谨慎、豪放不羁、叛逆反抗、忠义守信、领导支配、孤傲独立、乐观洒脱
- **29位好汉匹配**：通过欧几里得距离算法，匹配最接近的梁山好汉
- **完整性格报告**：每位好汉含人物速写、八维光谱、精神特质、人生寓言、现代映照
- **水墨江湖风 UI**：交互式山水背景、墨粒子效果、印章揭晓动画、Canvas 雷达图

## 在线体验

直接打开 `index.html` 即可使用，无需任何依赖。

## 部署到 GitHub Pages

1. Fork 或上传此仓库
2. 进入 Settings → Pages
3. Source 选择 `main` 分支，目录选 `/ (root)`
4. 保存后即可通过 `https://你的用户名.github.io/仓库名/` 访问

## 技术栈

- 纯 HTML + CSS + JavaScript（单文件，零依赖）
- Canvas API（交互式背景 + 雷达图）
- Google Fonts（Ma Shan Zheng + Noto Serif SC + Noto Sans SC）
